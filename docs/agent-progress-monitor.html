使用说明

用途：跟踪各 Agent 每日详细操作步骤的完成情况。在浏览器中打开本页即可查看；Agent 通过编辑本 HTML 文件更新状态。

更新规则：每完成一项本人当日任务后，编辑本文件：找到对应任务的 <input type="checkbox">，为其添加 checked 属性（即 <input type="checkbox" checked>）。可选在该 <li> 上增加 data-done="HH:mm" 或旁注完成时间。

更新范围：仅修改本人、当日所属区块（见各区块 data-agent、data-day）。勿改动其他 Agent 的区块。

任务列表与 digital_bank_poc_workplan.md 对齐；若 workplan 有变更，请同步更新本页。
